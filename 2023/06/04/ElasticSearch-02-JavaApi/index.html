问题

  1. Exception in thread “main” java.lang.NoClassDefFoundError: org/elasticsearch/client/Cancellable

    原因： elasticsearch-rest-high-level-client内部引用的是低版本，版本不一致

    解决：引入高版本，覆盖旧版本

  2. ERROR StatusLogger No log4j2 configuration file found. Using default configuration: logging only errors to the console. Set system property ‘org.apache.logging.log4j.simplelog.StatusLogger.level’ to TRACE to show Log4j2 internal initialization logging.

    原因：没有 log4j2 的配置文件

    解决：添加 log4j2 的配置文件

  3. Exception in thread “main” java.lang.NoClassDefFoundError: org/apache/commons/logging/LogFactory

    原因：缺少 commons-logging 包

    解决：引入 commons-logging 包依赖

  4. com/fasterxml/jackson/core/util/JacksonFeature

    原因：jackson-databind、jackson-core、jackson-annotations 版本不一致。（databind 中 包含 core、annotations 但是版本不一致）

    解决：将 jackson-databind、jackson-core、jackson-annotations 版本覆盖。

  5. jakarta/json/JsonException

    原因：Json 版本较低

    解决：指定版本覆盖
